# Stereotype `В«UnderConstructionВ»`
The stereotype `В«UnderConstructionВ»` marks a model element for which business or technical changes are planned.

The type of change can be specified using the tagged value `plannedChangeType`.


**Examples:**
* Preparing a future deprecation once the replacement mechanism is available.
* Introducing a new property that will become mandatory once the provider-side implementation is ready.
